# 『FleaScript』プロモーション映像制作仕様書（日常溶け込み×疑似ASMR版）

## 1. 演出コア・コンセプト
- **映像（ハイブリッド構成）**：お洒落な喫茶店での実写手元映像をベースに、アプリのUI操作やエフェクト部分にAI生成の洗練されたモーショングラフィックスを融合。プラットフォームのAI検閲を回避する。
- **データ制御**：画面に映る在庫データは、生活感を完全に排除したお洒落な「モックアップデータ（例：ハンドメイド革小物、洗練された洋書など）」に完全固定。
- **音響（後付けASMR）**：完全な静寂をベースとし、後ノセの高品質フリー素材（氷の音、カフェの環境音、UIクリック音）のみで構成。ローファイ・ビートを薄く敷く。

## 2. タイムライン＆構成案（12秒）

### シーン1：日常の静寂（0〜3秒）【実写＋フリーアセット音】
- **映像**：お洒落なカフェの窓際。実写でアイスティーのグラスをコトッと置く手元。
- **音響**：後付けの「カラン」と美しく響く氷のSE、および微かなカフェのざわめき（環境音）。
- **字幕**：出品文を考える15分。時給に換算すると……？

### シーン2：ハックの瞬間（3〜6秒）【実写＋AIエフェクト】
- **映像**：パステルカラーの『FleaScript』画面。実際の操作画面（実写）。指が「生成ボタン」をスマートに1タップする。
- **音響**：後付けの小気味良い電子クリック音（心地よいUI操作音）。
- **字幕**：私の時間は、私が守る。

### シーン3：戦果のプルーフ（6〜9秒）【実写（モックデータ固定）】
- **映像**：画面に美しいCanvasレポートが描画される。「生成：FleaScript」の透かし入り。
- **音響**：静かなローファイ・ビートのコードが優しく展開。
- **字幕**：今月、AIが生み出してくれた私の自由時間：115分。

### シーン4：CTA（9〜12秒）【AIモーショングラフィックス】
- **映像**：パステル背景に滑らかにフェードインする『FleaScript』の公式ロゴ。
- **音響**：余韻を残してフェードアウトする氷の音とビート。
- **字幕**：フリマの『めんどくさい』を消し去る魔法。FleaScript

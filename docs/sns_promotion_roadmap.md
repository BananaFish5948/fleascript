# FleaScript SNSプロモーション・ロードマップ

## 結論とコンセプト
公式アカウントによる「広告のばらまき」は現代のSNSアルゴリズムではスパム判定され、ユーザーにも敬遠される。
目指すべきは、ツール自身が宣伝を行う**「自己増殖型（UGC）バイラル・エコシステム」**の構築である。

## フェーズ別 実行計画

### Phase 1: 盤面の構築（Seed）
- **OGPの完全最適化**: X/Threadsでシェアされた際、FleaScriptのバナーが巨大なTwitter Cardとして魅力的に表示されるよう設定する（済）。
- **バイラル・コピーの策定**: 「フリマ出品がめんどくさい」というペイン（痛み）を突く、短く鋭いキャッチコピーを各SNSごとに用意する。
- **公式アカウントの「人間化」**: 単なる宣伝botではなく、フリマの裏技や梱包のコツを呟く「実利あるアカウント」として運用を開始する。

### Phase 2: UGCエンジンの着火（Ignition）
- **「月間レポート」機能のシェア促進**:
  - ユーザーに「今月の確定利益」の画像をワンタップでシェアさせる（Canvas生成画像の活用）。
  - 「自分の成果を自慢したい」という人間の承認欲求をハックし、画像内に「生成：FleaScript」の透かしを入れることで自然な広告とする。
- **リファラル報酬の可視化**:
  - シェアするだけで「+3枠ボーナス」が得られる仕組みを、制限モーダル（LimitModal）の最も目立つ位置で訴求する。

### Phase 3: プラットフォーム別 拡散戦略（コア動画アセットの投下）
手元にある「1〜2本の洗練された広告動画」は、無作為にばらまくのではなく、各プラットフォームの性質に合わせて**役割（フック、信頼構築、チュートリアル）**を変えて投下する。

- **TikTok / YouTube Shorts（認知のフック）**:
  - **戦略**: 動画をそのまま「ショート広告（縦型）」として出稿、またはオーガニック投稿。
  - **施策**: 梱包ASMRの背景で「商品説明はFleaScriptで1秒生成」という短いクリップとして流し、視覚的なインパクトだけでユーザーの指を止める。
- **X (旧Twitter)（信頼と拡散）**:
  - **戦略**: プロフィール画面の「固定ツイート」として動画を最上部に据える。
  - **施策**: UGC（ユーザーの「売れた！」というレポート投稿）がバズった際、プロフィールに飛んできた人が一番最初にその動画を見る「受け皿」にする。
- **Instagram（世界観とチュートリアル）**:
  - **戦略**: ハイライト（プロフィール下部の丸いアイコン）に動画を永続設置。
  - **施策**: 「今月〇〇万円稼げた主婦の裏技」といった静止画カルーセル投稿で興味を引き、プロフィールに飛んだユーザーに「実際の動き」を動画で納得させる。

### Phase 4: マイクロ・スポンサーシップ（Resonance）
- 数十万フォロワーのメガインフルエンサーではなく、フォロワー1,000〜5,000人規模の「副業アカウント」「節約主婦アカウント」に直接アプローチ。
- プレミアムプランを無償提供する代わりに、実際の使用感をストーリーズ等で紹介してもらう。

## ⚠️ 死角（リスクと対策）

- **スパム認定のリスク**: 全く同じ文面の自動ツイートを量産させると、Xのシャドウバン対象になる。シェア時のテキストは複数パターン用意するか、ユーザー自身に編集させる余白を残すこと。
- **期待値コントロール**: 広告で「何でも勝手に売れる魔法」と過剰に煽ると、実際の利用後にチャーン（離脱）率が跳ね上がる。「面倒な作業が消える」という実務的な価値に焦点を絞ること。
